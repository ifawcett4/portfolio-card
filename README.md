# Portfolio Card Site

Live site: https://www.irinafawcett.xyz

Hosted on GitHub Pages using the `gh-pages` branch.

---

## Development & Deployment

```bash
npm start
# runs the site locally at http://localhost:3000

npm run build
# creates a production build in /build (does NOT deploy)

npm run deploy
# builds the app and deploys it to GitHub Pages (updates live site)

## Common Issues

Double check that gh-pages is still at https://www.irinafawcett.xyz for the custom domain. It keeps switching to the default github one.
